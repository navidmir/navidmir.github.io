Machine Learning Hardware Accelerator

Hardware Accelerator for Image Classification Machine Learning Algorithm

UCLA ECE M216A: VLSI Systems
November 2021 - December 2021

This two-person class project was an RTL implementation of a neural network to perform number classification from images. The MNIST dataset was used to train the model. After the RTL was written in Verilog, we performed synthesis using Synopsys Design Compiler and we verified our design with ModelSim gate-level simulations. We spent the remaining time optimizing the design for minimum power and area and performed word-length simulation

100u x 20u NMOS IC Design

Design of digital and analog circuits with 100u x 20u NMOS

UCSB ECE 120B
April 2021 - June 2021

This group project was to design circuits with UCSB's teaching clean room NMOS process. Our group design a 2 by 2 binary multiplier, NAND gates, digital inverters, and common source amplifiers to be fabricated with 100u x 20u NMOS transistors. We designed the masks which were fabricated by the California NanoSystems Institute on campus. We then went through the four layer fabrication process in the clean room to fabricate the circuits.









Virtual Piano Teacher App

Android Piano Note Visualizer Application using FFT

UCSB ECE 150
May 2020 - June 2020

This solo project is an Android application written in Java that enables users to sample their piano playing and display the played notes on a piano GUI. The application uses the Fast-Fourier Transform (FFT) algorithm to produce the spectrum of the notes played to extract the frequency content of the sampled audio. The spectrum is then processed and the frequency components present are mapped to the key frequencies of a piano to display the notes played. The user can then record the piano GUI screen and save it. This application can be used by students who want to verify their note accuracy or piano teachers who want to record lessons for their students.



MOSFET Fabrication

Fabrication of n-Channel MOSFET from bare Silicon

UCSB ECE 120A
February 2020 - March 2020

This group lab project was to fabricate an n-MOSFET (n-Channel Metal-Oxide Semiconductor, Field Effect Transistor). Starting with bare, p-type doped Silicon wafers, we layered our samples with 4 total Masks. We used photolithography with a mask contact aligner to expose the photoresist on our samples and developed them in a base developer. We grew silicon dioxide for three different functions, one for the field, one to drive in our phosphorus n-type dopants for the source and drain regions, and one for the gate. We doped these souce and drain regions with phosphorus by first performing predeposition, and then performing a drive-in of the dopants with an oxide growth. To etch the oxide, we did wet-etching with hydrofluoric acid. Our last steps were to do aluminum deposition on the wafer with electron-beam evaporation and to perform liftoff to reveal our working transistors! The gate dimensions of the devices are from 100μm X 5μm to 400μm X 20μm.







Plasma Speaker

High-Voltage Plasma Arc Speaker

IEEE UCSB Chapter
October 2017 - May 2018

This project allowed for a music signal from any standard device to be played by a high-voltage plasma arc. I assembled three different circuits. The first circuit generates a PWM signal from the MIDI audio input. This PWM signal is then amplified in the second circuit. The final circuit, a MOSFET H-bridge, then converts the amplified DC signal to AC to allow for a transformer to step up the signal voltage and produce the arc at the corresponding music frequencies.

openSesame

Facial Recognition Smart-Lock

SB Hacks V
January 2019

This project uses OpenCV (Python Computer Vision Library) to match allowed users and automatically open a lock to a house or room, allowing key-less entry. This project was a submission to the SB Hacks V Hackathon at UC Santa Barbara where my team won the Best Security Hack Award sponsored by Arthrex, Inc. Users start by uploading a single image of each person allowed to enter on our website hosted on the Google Cloud Platform. When a guest wants to enter, they have a picture taken at the door which is then sent to our server that uses a machine learning computer vision algorithm to determine if the guest is a match with the allowed guests. The server then sends a response to a Raspberry Pi located at the door. The Raspberry Pi interprets this response and then sends it to an Arduino which then can operate a servo that locks/unlocks the door and displays messages to the guests about their status. I designed and put together hardware and worked on software integration. Specifically, I made a Raspberry Pi circuit that runs JavaScript code to access our server on the Google Cloud Platform for a facial recognition program response. The Raspberry Pi then converts the response to a digital signal to have an Arduino turn a lock using servo motors and display messages on an LCD.
