Plasma Speaker

High-Voltage Plasma Arc Speaker

IEEE UCSB Chapter

This project allowed for a music signal from any standard device to be played by a high-voltage plasma arc. I assembled three different circuits. The first circuit generates a PWM signal from the MIDI audio input. This PWM signal is then amplified in the second circuit. The final circuit, a MOSFET H-bridge, then converts the amplified DC signal to AC to allow for a transformer to step up the signal voltage and produce the arc at the corresponding music frequencies.

openSesame

Facial Recognition Smart-Lock

SB Hacks V

This project uses OpenCV (Python Computer Vision Library) to match allowed users and automatically open a lock to a house or room, allowing key-less entry. This project was a submission to the SB Hacks V Hackathon at UC Santa Barbara where my team won the Best Security Hack Award sponsored by Arthrex, Inc. Users start by uploading a single image of each person allowed to enter on our website hosted on the Google Cloud Platform. When a guest wants to enter, they have a picture taken at the door which is then sent to our server that uses a machine learning computer vision algorithm to determine if the guest is a match with the allowed guests. The server then sends a response to a Raspberry Pi located at the door. The Raspberry Pi interprets this response and then sends it to an Arduino which then can operate a servo that locks/unlocks the door and displays messages to the guests about their status. I designed and put together hardware and worked on software integration. Specifically, I made a Raspberry Pi circuit that runs JavaScript code to access our server on the Google Cloud Platform for a facial recognition program response. The Raspberry Pi then converts the response to a digital signal to have an Arduino turn a lock using servo motors and display messages on an LCD.

LED Tetris

32 x 32 LED Panel Tetris Game Console

UCSB ECE 5

This project allows users to play Tetris on a vibrant LED panel. I wrote the Tetris game code on an Arduino Mega, which interfaces with the LED panel via a library that controls each LED individually. I also integrated a joystick to give an arcade-style feel to the user.

MicroMouse Prototype

Autonomous Maze-Solving Robot Prototype

IEEE UCSB Chapter

This robot prototype is designed to autonomously solve and traverse a maze in the fastest amount of time. I constructed a wood base prototype to test a design. I assembled the circuits which used infrared distance sensors, motor drivers, and stepper motors. I also wrote microcontroller code that allowed for sensor data to be read and converted to distance measurements and code that allowed for driving the bipolar stepper motors.

Machined Lightsaber

Machined Aluminum Lightsaber Hilt

UCSB Machine Shop Club

I created a lightsaber with a machined aluminum handle through the UC Santa Barbara Machine Shop Club. I designed CAD specifications of the handle from scratch using SolidWorks. I then made the handle by lathing aluminum stock based off of the specifications of my CAD design.
