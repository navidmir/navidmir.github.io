Plasma Speaker

High-Voltage Plasma Arc Speaker

IEEE UCSB Chapter

This project allowed for a music signal from any standard device to be played by a high-voltage plasma arc. I assembled three different circuits: a signal generator circuit that converts a MIDI signal to a PWM signal, a signal amplifier circuit, and a MOSFET H-bridge circuit that then converts the amplified DC signal to AC to allow for a transformer to step up the signal voltage and produce the arc.

openSesame

Facial Recognition Smart-Lock

SB Hacks V

This project uses OpenCV (Python Computer Vision Library) to match allowed users and automatically open a lock to a house or room, allowing key-less entry. Users start by uploading a single image of each person allowed to enter on our website hosted on the Google Cloud Platform. When a guest wants to enter, they have a picture taken at the door which is then sent to our server that uses machine learning and computer vision to determine if the guest is a match with the allowed guests. The server then sends a response to a Raspberry Pi that acts as its own server. The Raspberry Pi interprets this response and then sends it to an Arduino which then can operate a servo that locks/unlocks the door and displays messages to the guests about their status. I designed and put together hardware and worked on software integration. Specifically, I made a Raspberry Pi circuit that runs JavaScript code to access our server on the Google Cloud Platform for a facial recognition program response. The Raspberry Pi then converts the response to a digital signal to have an Arduino turn a lock using servo motors and display messages on an LCD.

MicroMouse Prototype

Autonomous Maze-Solving Robot Prototype

IEEE UCSB Chapter

This robot prototype is designed to autonomously solve and traverse a maze in the fastest amount of time. I constructed a wood base prototype to test a design. I assembled the circuits which used infrared distance sensors, motor drivers, and stepper motors. I also wrote microcontroller code that allowed for sensor data to be read and converted to distance measurements and code that allowed for driving the bipolar stepper motors.
